# 🚗 GigaShop

Uma landing page moderna para venda de carros com carrinho de compras e integração com WhatsApp.

---

## 📸 Preview

![preview](./assets/logo.png)

---

## 🚀 Funcionalidades

- 🎥 Hero com vídeo de fundo
- 🛒 Carrinho de compras dinâmico
- ➕ Controle de quantidade de produtos
- 💾 Persistência com localStorage
- 📱 Integração com WhatsApp para finalizar pedidos
- 🎨 Interface moderna com modal interativo

---

## 🧠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
